# MarkDown
* 일반 텍스트 기반의 경량 마크업 언어
* 보기좋은 문서를 보여주기 위하여 문서에 추가되는 정보  (문서 포맷 지정, 표, 그림, 문서간 링크...)
* 일반 텍스트로 서식이 있는 문서 작성 시 사용
* 문법이 쉽고 간단한 것이 특징


하나하나 알아보자

### 헤더표시

* 문법

```
# 주제1
## 주제1
### 주제1
#### 주제1
##### 주제1
###### 주제1
####### 주제1
```
